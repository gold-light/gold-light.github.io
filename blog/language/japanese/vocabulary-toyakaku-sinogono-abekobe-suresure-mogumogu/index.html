とやかく, 四の五の, あべこべ, すれすれ, もぐもぐ

とやかく link

  • 의미: 이러쿵저러쿵, 이러니저러니

예문 link

  • 他人のことをとやかく言うな。

四の五の (しのごの) link

  • 의미: 이러쿵저러쿵, 이러니저러니

예문 link

  • 四の五の言わないで、早く勉強しなさい。

あべこべ link

  • 의미: 반대로, 거꾸로, 뒤바뀐

예문 link

  • 先生が学生にあべこべに教えられる。

すれすれ link

  • 의미: 아슬아슬함, 닿을락말락함, 빠듯함

예문 link

  • 試験にすれすれのところで合格しました。
  • 出発時間にすれすれで間に合いました。

もぐもぐ link

  • 의미: 입을 벌리지 않고 씹는 모양의 우물우물, 입 안에서 중얼거리는 모양의 어물어물

예문 link

  • 彼はいつももぐもぐ(と)食べている。
  • 彼女が何かもぐもぐ(と)言う。
