히트2 (Hit 2) 아이템 분해하는 방법

아이템 분해 기능 link

넥슨이 얼마 전에 공개한 모바일 RPG게임 히트2를 플레이 하다보니 아이템 분해 기능을 찾지 못해 살짝 고생했습니다.

보통 아이템을 분해하려면 해당 아이템을 선택하고 분해 버튼이나 메뉴를 선택하면 분해를 할 수 있게 되는데, 히트2에는 그런 UI의 편의성이 전혀 없습니다.

겨우겨우 아이템 분해하는 방법을 찾았는데, 혹시나 저 처럼 고생하시는 분이 있을까봐 정리해보 았습니다.

분해 방법 link

일단 가방을 선택하고 아이템 리스트 아래쪽에 보시면 아래 스샷처럼 아이템 분해 아이콘이 있습니다.

사실 잘 보이지도 않고, 마우스를 올려보기 전까지는 아이템 분해 버튼인지 잘 확인도 안되는게 좀 안타깝습니다.

아무튼 [아이템 분해] 버튼을 클릭하면 분해할 수 있는 아이템 리스트가 나타나는데 분해할 아이템을 전부 클릭해서 선택하고 [분해] 버튼을 클릭하면 됩니다.

자동 선택 설정 link

아이템 분해할 때는 미리 지정한 아이템 종류는 자동으로 선택되게 할 수 있는데, 아래와 같이 [자동 설정]에서 미리 지정할 수 있습니다.

게임 소개 link

히트2에서 공개한 게임소개를 가져와봤는데 한번 살펴보겠습니다.

1. 당신의 부름에 응답한 히트의 영웅들 link

하이엔드 비쥬얼로 재탄생한 키키와 새로운 동료들!
다음 세대를 위해 준비된 광활한 오픈 월드에서 더욱 생동감 넘치는 비주얼로 돌아온 히트를 만나보세요.

2. 시그니쳐 스킬로 재탄생한 히트의 오리진 액션 link

각각의 매력이 뚜렷한 6종의 클래스가 펼치는 전투의 향연!
광활한 필드 곳곳을 수놓는 화려한 스킬과 치열한 전투는 히트2에서도 계속됩니다.

3. 서버 오픈 시점부터 시작되는 치열한 공성전 link

누구나 참여할 수 있고, 언제든 시작되는 명예로운 전투!
모험가님의 협동과 경쟁에 따라 특별한 이벤트와 함께 왕좌의 주인을 결정할 공성전이 바로 시작됩니다.

4. 전장을 뒤흔드는 조율자의 제단 link

당신이 선택한 필드와 월드의 규칙!
그 규칙 위에서 새롭게 쓰여지는 월드의 역사! 다채롭게 변화하는 전투 양상과 레이드를 경험해보세요.

5. 붉은 회오리가 불면 눈을 뜨는 에덴의 변이체 link

시시각각 에르바덴을 위기에 빠뜨리는 미지의 공격!
한 순간도 긴장을 놓을 수 없는 월드의 변화와 위협에 맞서 위기에 빠진 세계를 구원해주세요.
