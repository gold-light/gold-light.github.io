[인슈라오디드] 최저 사양 : GTX 750Ti ? 1050Ti ?

개요 link

요즘 [팰 월드(PalWorld)]에 이어서 화제가 되고 있는 오픈 월드 생존 RPG 게임인 [인슈라오디드(Enshrouded)]를 구매 후 플레이 해보려고 했는데 실행이 되지 않길래 대체 최소 사양이 어느 정도이길래 [디아블로 4(Diablo 4)], [팰 월드(PalWorld)] 모두 잘 실행되는데 [인슈라오디드(Enshrouded)]만 실행이 안되는 걸까 싶어서 최소 사양을 확인해봤습니다.

시스템 요구 사항 link

[스팀] 소개 페이지에 나왔있는 [인슈라오디드(Enshrouded)]의 시스템 요구 사항을 확인해보면 다음과 같습니다. 이 중에서 눈에 띄는 것은 [req. 6GB VRAM] 이 부분으로 그래픽카드 메모리를 많이 요구하는 것을 알 수 있었습니다.

물론 이 사양은 개발사에서 안정적으로 플레이 하기 위한 최소, 권장 사양이므로 이 보다 낮은 사양에서도 플레이 가능한 경우도 많기 때문에 구매 후 플레이 해보려고 했습니다.

플레이 PC 사양 link

제가 사용하는 PC는 아주 오래전에 구매했었는데, 중간에 메모리랑 그래픽 카드만 업그레이드 한번 했었습니다. 이 PC로 [디아블로3], [디아블로2 레저렉션], [디아블로 이모탈], [디아블로4], [히트2], [블레스 언리쉬드], [문명6], [로스트아크], [팰 월드], [파이오니어스 오브 파고니아] 등을 플레이했습니다.

  • 운영 체제: 64비트 Windows 10
  • 프로세서: Intel Core i5-4570 CPU 3.20GHz
  • 메모리: 16GB RAM
  • 그래픽: NVIDIA GeForce GTX 750 Ti
  • 디스크: WD SSD 500G

실행 불가 메시지 link

게임을 실행하면 아래와 같이 그래픽 메모리가 너무 낮습니다. 라는 오류 메시지가 뜨면서 실행이 되지 않았습니다. 그나마 최근에 패치가 되면서 오류 메시지가 정확히 나타는 것이지, 처음에는 검은 화면에 마우스 포인트만 보이다가 게임이 종료되는 당황스러운 상황만 지속되었습니다.

GeForce GTX 750 Ti 정보 link

그래서 현재 그래픽 카드인 [GeForce GTX 750 Ti]의 정보를 확인해보았습니다.
확인해봤더니 아래 스샷처럼 [전용 비디오 메모리]가 [2GB] 인 것을 알 수 있었습니다. 즉, 공식 소개 페이지에 나와있던 시스템 요구사항에서 확인했 듯이 그래픽카드의 메모리가 매우 중요하다는 것을 다시 한번 확인할 수 있었습니다.

GeForce GTX 1050 Ti 정보 link

플레이 할 수 없으니 환불을 해야 하나 고민하다가 회사에 있는 PC는 [GTX 1050 Ti] 그래픽카드가 장착되어 있으므로 혹시 가능할까 싶어서 그래픽카드 정보를 확인해보았더니 [전용 비디오 메모리]가 [4GB]였습니다. 다행히도 [GTX 1050 Ti]에서는 무사히 실행되었고, 새 그래픽카드를 구매하기 전까지 당분간은 회사에서 플레이 해보기로 했습니다.

플레이 화면 link

[GTX 1050 Ti] 그래픽카드에서 플레이 해봤는데 이런 화면이었습니다.

결론 link

[인슈라오디드(Enshrouded)]을 플레이 하기 위한 최소 사양은 [GTX 1050 Ti - 4GB VRAM]인데, 조금 더 안정적으로 플레이 하기 위해서는 [GTX 1060] 이상에 가능하면 6GB VRAM이 장착된 그래픽카드를 사용하는 것이 좋을 듯합니다.
