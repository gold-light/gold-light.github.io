[스팀] 액션 RPG 게임 기대작 [바벨탑: 혼돈의 생존자들] Demo

개요 link

이번에 소개할 스팀 기대작은 [뱀파이어 서바이버즈]와 [디아블로]의 서바이벌 요소와 루팅 RPG가 결합된 싱글 플레이어 서바이벌 액션 RPG 게임 [바벨탑: 혼돈의 생존자들 (Tower of Babel: Survivors of Chaos)]입니다.

현재 Demo버전이 2025년 1월 22일에 오픈된다고 하는데, 저는 [디아블로]와 비슷하면서 가볍게 즐길 수 있을 것 같아서 어떤 게임인지 정리해봤습니다.

특징 link

  • 아름다운 핸드 드로잉 2D 그래픽 다크 판타지
  • 서바이벌 전투 액션
  • 다양한 스킬과 능력로 자신만의 빌드업
  • 수백개의 아이템 루팅
  • 랜덤 옵션의 희귀한 고유 아이템
  • 자원을 모으고 업그레이드

게임 화면 link

인벤토리 link

인벤토리와 아이템 정보 창은 디아블로와 매우 비슷해보여서 금방 익숙해지겠네요.

아이템 가방과 캐릭터 link

아이템 가방과 캐릭터 창은 디아블로랑 조금 다르면서 더 편리한 것 같기도 합니다.

스킬 사용 link

스킬은 매우 화려해서 보는 맛이 좋을 합니다.

개발사 link

개발과 서비스는 모두 [NANOO(나누컴퍼니)]에서 하고 있는데, 이전에는 모바일 게임을 주로 만들던 개발사라고 합니다. 대표작으로는 [다크 소드], [안녕 산토리니], [슈퍼 애로우 키우기] 등이 있습니다.

시스템 요구 사항 link
