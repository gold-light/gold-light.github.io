디아블로4 고행 진입 퀘스트 - 몰락자의 사원 엘리아스 클리어 영상과 아이템 정보

개요 link

디아블로4 (Diablo 4) 얼음파편 냉기 법사로 고행 진입을 위한 퀘스트 - [몰락자의 사원] 클리어한 영상과 착용한 아이템 정보들을 정리해보았습니다.
[몰락자의 사원]은 [메마른 평원] 지역에 있는데, 보스 몬스터 [증오의 잔해 엘리아스]를 처치하면 완료됩니다.

플레이 PC 사양 link

제가 사용하는 PC는 아주 오래전에 구매했었는데, 중간에 메모리랑 그래픽 카드만 업그레이드 한번 했었습니다. 이 PC로 디아블로3, 디아블로2 레저렉션, 디아블로 이모탈, 히트2, 블레스 언리쉬드, 문명6, 로스트아크 등을 플레이했습니다.

  • 운영 체제: 64비트 Windows 10
  • 프로세서: Intel Core i5-4570 CPU 3.20GHz
  • 메모리: 16GB RAM
  • 그래픽: NVIDIA GeForce GTX 750 Ti
  • 디스크: WD SSD 500G
  • Xbox 4세대 게임패드 카본 블랙

몰락자의 사원 전체 클리어 영상 link

이 영상은 던전 들어가서 일반 몬스터를 처치하는 부분부터, 보스 몬스터 [증오의 잔해 엘리아스]를 처치하고 고행 진입하기까지 전체 영상입니다.

보스 엘리아스 전투 전체 영상 link

컨트롤이 워낙 서툰데다 키보드가 아닌 엑스박스 게임패드로 처음 플레이 해보다 보니 보스 몬스터 엘리아스와 전투하면서 2번이나 실패를 했는데, 이 영상은 실패한 것 포함해서 보스와 전투 영상 전체 버전입니다.

보스 엘리아스 전투 편집 영상 link

이 영상은 보스와 전투하면서 성공한 3번째 전투만 편집한 영상입니다.

최대 데미지 link

최대 데미지가 24,262 정도 나왔는데, 아직은 갈길이 멀군요.

1차 시도 - 실패 link

1차 시도에서 보스 엘리아스의 체력을 절반 넘게 깍았는데, 화면 전체에 떨어지는 공격을 예상 못하고 실패했습니다.

2차 시도 - 실패 link

2차 시도에서도 같은 패턴의 공격 유형에 또 당했습니다. 이런 ㅠ.ㅠ

3차 시도 - 성공 link

이번에는 바짝 정신 차려서 해봤습니다.

  • 첫번째 비틀거림(그로기) 상태입니다.
  • 두번째 비틀거림(그로기) 상태입니다.
  • 그렇게 공격하고 나니 보스 체력이 얼마 남지 않아서 닥공으로 맞붙었습니다.
  • 보스 엘리아스를 드디어 처치했습니다.

고행 단계 이동 link

기다리고 기다리던 고행 단계로 드디어 전환했습니다.
