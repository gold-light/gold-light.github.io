디아블로4 1.1.0a 시즌 1 패치 야만용사 관련 내역 총정리

개요 link

디아블로4에서 7월 19일에 시즌 1 시작에 대비한 1.1.0a 패치가 진행되었습니다. 한번에 정리하기 힘들 정도로 엄청만 양의 패치 내역이기도 하고, 이래저래 말도 많은 패치였기에 한번에 알아보기 쉽게 직업별로 정리해보려고 합니다. 우선 야만용사(바바리안, BarBarian) 관련 패치 정보를 정리해보겠습니다.

⁃ 디아블로4 홈페이지 전체 패치 노트 확인하기

고유 아이템 link

일반 link

  • 아발리온 - 라이칸더의 창(우버 고유 지팡이 – 세계 단계 4): 정예 괴물 처치 시 10~20초간 무작위 제단 효과를 획득합니다. 이 효과는 30초에 한 번씩 발동됩니다.

야만용사 link

  • 푸른 서슬(고유 도검): 행운의 적중 - 핵심 기술이 최대 20% 확률로 적들을 3초 동안 빙결시키고 0.75~1.5의 냉기 피해를 줍니다.
푸른 서슬 고유 아이템의 피해 효과는 고정되어 있으며 위력에 따라 증가합니다. 예를 들어, 아이템 위력이 820인 경우 푸른 서슬은 3,359~6,718의 피해를 줍니다.

신규 전설 위상 link

일반 link

  • 용맹(보조 위상): 주변의 적이 5명 이상일 때 2~4초간 기절시킵니다. 이 효과는 20초에 한 번씩 발동됩니다.
  • 비겁자(기동력 위상): 감속되거나 오한을 느끼는 적에게서 멀어질 때 이동 속도가 20~40% 증가합니다.

야만용사 link

  • 선조의 돌진(공격 위상): 돌진이 4명의 선조를 부릅니다. 선조들은 돌진하여 원래 피해의 50~100%만큼 피해를 줍니다.

버그 수정 link

게임플레이 link

야만용사 link

  • 야만용사 직업 퀘스트 전투의 달인에서 처치해야 할 적의 수가 표시되지 않던 문제를 수정했습니다.
  • 전장의 함성을 외쳐서 강철 소용돌이를 3번 이상 빠르게 시전하던 문제를 수정했습니다.
  • 광폭한 찢기, 해골 격파자, 빈혈 위상을 중첩할 시 플레이어가 지나치게 큰 피해를 주던 문제를 수정했습니다.
  • 선조의 메아리 위상이 적용된 선조의 망치가 일으킨 진동의 가장자리가 의도한 것보다 많은 피해를 주던 문제를 수정했습니다.

일반 link

  • 우두머리와 전투 시 플레이어가 비정상적으로 순간이동하던 문제를 수정했습니다.
  • 서쪽 길의 상자에 발생하는 무한 상호작용 문제를 수정했습니다.
  • 일부 상황에서 에리만 NPC와 상호작용하지 못하던 문제를 수정했습니다.
  • 파괴할 수 있는 물체를 통과하려고 할 때 부수지 못하던 문제를 수정했습니다.
  • 플레이어가 특정 위치에 자리하면 우두머리에게 피해를 입지 않던 사례를 수정했습니다.
  • 제련된 유물에 상호작용하지 못하던 문제를 수정했습니다.
  • 악몽 던전에서 냉기, 출혈, 화상, 또는 타락 속성의 괴물이 냉기 강화 접사를 가진 채 1회 공격하면 플레이어가 빙결되던 문제를 수정했습니다.
  • 높은 세계 단계에서 냉기 강화 괴물이 플레이어를 의도한 것보다 더 자주 빙결시키던 문제를 수정했습니다.
  • 전리품이 무한히 떨어지는 궤짝이 생성되던 문제를 수정했습니다.

밸런스 업데이트 link

야만용사 link

기술 link

  • 난폭한 이중 타격의 취약 지속 시간이 1초에서 2초로 증가했습니다.
  • 맹렬한 지각 변동의 중첩당 추가 피해가 3%에서 5%로 증가했습니다.
  • 강력한 돌진의 재사용 대기시간 감소 효과가 대상 3명으로 제한되고, 최대 9초까지 재사용 대기시간이 감소합니다.
  • 전술적 발 구르기의 분노 획득량이 25에서 40으로 증가했습니다.
  • 강화된 결정타의 우두머리에 대한 추가 피해가 100%에서 150%로 증가했습니다.
  • 강화된 강철 손아귀의 취약 지속 시간이 2.5초에서 3초로 증가했습니다.
  • 강화된 파열 피해가 30%에서 45%로 증가했습니다.
  • 전사의 파열 공격 속도 보너스가 20%에서 30%로 증가하고, 지속 시간이 4초에서 5초로 증가했습니다.
  • 투사의 파열 회복력이 최대 생명력의 15%에서 18%로 증가합니다.

지속 효과 link

  • 무력화는 건강 상태의 적만 감속시키고, 감속 속도는 10/20/30%에서 20/40/60%로 증가합니다.
  • 취약 상태에 대한 뼈 가르기의 출혈 추가 피해가 6/12/18%에서 5/10/15%로 감소합니다.

정복자 link

  • 진노 문양의 보너스가 핵심 기술뿐만 아니라 모든 피해에 적용됩니다.

위상 link

  • 가차없는 광전사의 위상 광폭화 지속 시간이 1초에서 2초로 증가했습니다.
  • 전투의 광기 위상 광폭화 지속 시간이 12.5초에서 23.5초로 증가했습니다.
  • 난투가의 위상 피해가 11.3에서 1.51.8로 증가했습니다.
  • 불타는 분노의 위상 피해가 0.080.14에서 0.220.28로 증가했습니다.
  • 철갑 전사의 위상 피해 감소가 1525%에서 1828%로 증가했습니다.
  • 갈증 해소의 위상 행운의 적중 확률이 3050%에서 4060%로 증가했습니다.
  • 선조의 메아리의 위상 행운의 적중 확률이 4050%에서 5060%로 증가했습니다.
  • 무기 전문가 위상 행운의 적중 확률이 32~50%에서 37%~55%로 증가했습니다.

아이템 link

  • 고르의 파멸적인 손 보호구는 이제 2초마다 또는 소용돌이 종료 후 폭발하고, 적중 제한이 없어졌습니다.
  • 심홍색의 전장 출혈 추가 피해가 10%에서 15%로 증가했습니다.

일반 link

위상 link

  • 불복의 위상의 추가 방어도가 25%~50%에서 15%~30%로 감소하고, 최대 중첩은 100에서 60으로 감소했습니다.
  • 응보의 위상이 기절한 적에게 주는 추가 피해가 2040%에서 1020%로 감소했습니다.
  • 착취자의 위상이 저지 불가 적에게 주는 추가 피해가 2050%에서 2040%로 감소했습니다.
  • 별빛 위상 자원 회복량이 1020에서 2040으로 증가했습니다.

아이템 link

  • 도살자의 식칼 감속 효과가 40~75%에서 61-75%로 증가했습니다.

  • 드랍률 조정:

    • 악몽 던전에서 선조 아이템 및 신성 아이템을 획득할 추가 확률이 +5%에서 +10%로 증가했습니다.
    • 악몽 던전과 지옥물결 이외에서 정예 괴물이 추가 장비 아이템을 떨어뜨릴 확률 20%가 제거되었습니다.
    • 악몽 던전과 지옥물결의 정예 괴물이 추가 장비 아이템을 떨어뜨릴 확률이 10%만큼 증가했습니다.
    • 이제 악몽 던전을 완료하면 50% 확률로 두 번째 전설 아이템을 획득할 수 있습니다.
    • 악몽 던전 완료 시 얻는 희귀 아이템이 1개에서 3개로 변경되었습니다.

괴물 link

  • 파티에 추가되는 플레이어당 증가하는 괴물 HP가 85%에서 100%로 조정되었습니다.

기타 link

  • 탐욕의 신단 개수를 조정하여 게임 전반의 효율을 개선했습니다.
  • 이제 야외 우두머리가 물약을 더 자주 떨어뜨립니다.
  • 이제 상위 재료로 정제할 수 있는 자원의 양에 제한이 없습니다.

아이템 접사 link

여러 빌드에 쉽게 적용할 수 있도록 다음 접사의 가용성을 늘렸습니다.

  • 제어 방해 지속 시간 감소: 이제 바지에 나타납니다.

  • 보호막 생성량: 이제 모든 직업에 나타납니다. 또한 게임 후반에 획득하기 시작합니다.

  • 보호막이 있을 때 행운의 적중 확률: 이제 모든 직업에 나타납니다. 투구에 적용 시 ~12%까지, 목걸이 또는 보조 무기에 적용 시 ~20%까지 감소합니다.
