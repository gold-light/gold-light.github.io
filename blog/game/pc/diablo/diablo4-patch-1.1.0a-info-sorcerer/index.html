디아블로4 1.1.0a 시즌 1 패치 원소술사 관련 내역 총정리

개요 link

디아블로4에서 7월 19일에 시즌 1 시작에 대비한 1.1.0a 패치가 진행되었습니다. 한번에 정리하기 힘들 정도로 엄청만 양의 패치 내역이기도 하고, 이래저래 말도 많은 패치였기에 한번에 알아보기 쉽게 직업별로 정리해보려고 합니다. 우선 원소술사 관련 패치 정보를 정리해보겠습니다.

⁃ 디아블로4 홈페이지 전체 패치 노트 확인하기

고유 아이템 link

일반 link

  • 아발리온 - 라이칸더의 창(우버 고유 지팡이 – 세계 단계 4): 정예 괴물 처치 시 10~20초간 무작위 제단 효과를 획득합니다. 이 효과는 30초에 한 번씩 발동됩니다.

원소술사 link

  • 마력의 눈(고유 마법봉): 자원 소모 없이 순간이동 마법부여의 효과를 얻습니다. 순간이동 마법부여를 사용해 피하면 무작위 위치로 순간이동합니다.

신규 전설 위상 link

일반 link

  • 용맹(보조 위상): 주변의 적이 5명 이상일 때 2~4초간 기절시킵니다. 이 효과는 20초에 한 번씩 발동됩니다.
  • 비겁자(기동력 위상): 감속되거나 오한을 느끼는 적에게서 멀어질 때 이동 속도가 20~40% 증가합니다.

원소술사 link

  • 타오르는 수호물(공격 위상): 마나를 200~100 소모하면 다음 화염벽이 자원 소모 없이 시전되고, 날아오는 소형 투사체를 파괴합니다.

원소술사 link

  • 화염탄 마법 부여 효과가 연소 핵심 지속 효과에 적용되지 않던 문제를 수정했습니다.
  • 원소 조절로 생성한 지속 효과 중첩이 의도치 않게 부분적으로 초기화되던 문제를 수정했습니다.
  • 냉대를 시전하면 얼음 파편 마법 부여 효과가 발동되지 않던 문제를 수정했습니다.
  • 비틀거리는 우두머리에게 제어의 위상 피해 보너스가 3번 적용되던 문제를 수정했습니다.
  • 양손 무기나 목걸이에서 중력의 위상이 약하게 나타나던 문제를 수정했습니다.
  • 연쇄 번개 마법 부여 효과가 시각적인 추적 보너스를 표시하지 않는 문제와 갑자기 사라지던 문제를 수정했습니다.
  • 소각 시전 시 일정하게 마나를 소모하면서도 추가로 30 마나를 즉시 소모하던 문제를 수정했습니다. 이제 소각은 시전 중에만 정상적으로 마나를 소모할 것입니다.

일반 link

  • 우두머리와 전투 시 플레이어가 비정상적으로 순간이동하던 문제를 수정했습니다.
  • 서쪽 길의 상자에 발생하는 무한 상호작용 문제를 수정했습니다.
  • 일부 상황에서 에리만 NPC와 상호작용하지 못하던 문제를 수정했습니다.
  • 파괴할 수 있는 물체를 통과하려고 할 때 부수지 못하던 문제를 수정했습니다.
  • 플레이어가 특정 위치에 자리하면 우두머리에게 피해를 입지 않던 사례를 수정했습니다.
  • 제련된 유물에 상호작용하지 못하던 문제를 수정했습니다.
  • 악몽 던전에서 냉기, 출혈, 화상, 또는 타락 속성의 괴물이 냉기 강화 접사를 가진 채 1회 공격하면 플레이어가 빙결되던 문제를 수정했습니다.
  • 높은 세계 단계에서 냉기 강화 괴물이 플레이어를 의도한 것보다 더 자주 빙결시키던 문제를 수정했습니다.
  • 전리품이 무한히 떨어지는 궤짝이 생성되던 문제를 수정했습니다.

기술 link

  • 강화된 번개 줄기 피해가 24%에서 30%로 증가하고, 기술 등급 혜택을 받습니다.
  • 파괴적인 번개 줄기 피해 감소가 20%에서 25%로 증가합니다.
  • 파괴적인 소각 피해 감소가 20%에서 25%로 증가합니다.
  • 얼음 칼날 시전 후 지연 시간이 짧아져 다음 동작을 더 빠르게 수행할 수 있습니다.
  • 화염탄의 기본 피해가 10%에서 18%로 증가했습니다.
  • 마법사의 화염벽 마나 재생량이 최대 25%에서 35%로 증가합니다.
  • 마법사의 눈보라 핵심 기술의 마나 소모량 감소가 10%에서 20%로 증가합니다.
  • 최고의 냉대 재사용 대기시간 감소가 50%에서 100%로 증가했습니다.

지속 효과 link

  • 원소 지배가 핵심 기술뿐만 아니라 숙련 기술에 영향을 미칩니다.
  • 강력한 수호 지속 시간이 3초에서 9초로 증가했습니다.
  • 피해 감소 보호막 지속 시간이 2초에서 3초로 증가했습니다.
  • 원소 조절의 초당 피해 감소가 1/2/3%에서 3/6/9%로 증가했습니다.
  • 마나 보호막 피해 감소가 5/10/15%에서 7/14/21%로 증가했습니다.
  • 걸신들린 불길 추가 극대화 피해가 10/20/30%에서 7/14/21%로 감소했습니다.
  • 걸신들린 불길의 이동 불가 대상에 대한 추가 극대화 피해가 25/50/75%에서 10/20/30%로 감소했습니다.
  • 불타는 과도 전류의 마나 재생량이 10/20/30%에서 15/30/45%로 증가했습니다.

정복자 link

  • 파괴 문양이 핵심 기술뿐만 아니라 모든 피해에 적용됩니다.

마법부여 link

  • 벼락불 마법부여로 짜릿한 에너지를 생성할 확률이 10%에서 14%로 증가했습니다.
  • 얼음 보주 마법부여가 시전될 확률이 20%에서 30%로 증가했습니다.
  • 소각 마법부여 재사용 대기시간이 18초에서 14초로 감소했습니다.
  • 히드라 마법부여의 마나 소모량이 300에서 200으로 감소했습니다.

전설 위상 link

  • 효율의 위상 마나 소모량 감소가 1020%에서 1525%로 증가하고, 숙련 기술에도 영향을 미칩니다.
  • 아마겟돈의 위상 피해가 0.2050.25에서 0.350.4로 증가했습니다.
  • 산산조각 난 별의 위상 피해가 0.150.2에서 0.20.25로 증가했습니다.
  • 구렁이의 위상 히드라 지속 시간 감소가 3020%에서 2010%로 감소했습니다.
  • 빙하의 위상 얼음 가시가 빙결된 대상에게 주는 추가 피해는 25%에서 15%로 감소했습니다.
  • 빙하의 위상 얼음 가시 피해가 0.20.3에서 0.250.35로 증가했습니다.
  • 얼어붙은 동토의 위상 얼음 가시 피해가 0.250.35에서 0.30.4로 증가했습니다.
  • 얼어붙은 흔적의 위상 얼음 가시 피해가 0.20.3에서 0.250.35로 증가하고, 오한이 10%에서 15%로 증가했습니다.
  • 파고드는 추위의 위상이 빙결된 적을 취약하게 만들 확률은 2535%에서 3040%로 증가했습니다.
  • 둘러싸인 위상의 초당 생명력과 마나 회복력이 1020%에서 2030%로 증가했습니다.
  • 안정적인 위상 자원 소모 없는 시전 확률이 510%에서 712%로 증가했습니다.
  • 갈라지는 마력의 위상 추가 번개 창 생성 확률이 1120%에서 4150%로 증가했습니다.
  • 끊어지지 않는 끈의 위상 연쇄 번개가 추가될 확률이 2535%에서 3040%로 증가했습니다.

아이템 link

  • 에수의 가보 극대화 확률이 이동 속도 보너스의 15~25%에서 20-30%로 증가했습니다.
  • 에사도라의 넘실거리는 카메오가 번개 회오리를 생성할 확률이 10%에서 15%로 증가했습니다.

위상 link

  • 별빛 위상 자원 회복량이 1020에서 2040으로 증가했습니다.

아이템 link

  • 도살자의 식칼 감속 효과가 40~75%에서 61-75%로 증가했습니다.

  • 드랍률 조정:

    • 악몽 던전에서 선조 아이템 및 신성 아이템을 획득할 추가 확률이 +5%에서 +10%로 증가했습니다.
    • 악몽 던전과 지옥물결 이외에서 정예 괴물이 추가 장비 아이템을 떨어뜨릴 확률 20%가 제거되었습니다.
    • 악몽 던전과 지옥물결의 정예 괴물이 추가 장비 아이템을 떨어뜨릴 확률이 10%만큼 증가했습니다.
    • 이제 악몽 던전을 완료하면 50% 확률로 두 번째 전설 아이템을 획득할 수 있습니다.
    • 악몽 던전 완료 시 얻는 희귀 아이템이 1개에서 3개로 변경되었습니다.

괴물 link

  • 이제 야외 우두머리가 물약을 더 자주 떨어뜨립니다.
  • 이제 상위 재료로 정제할 수 있는 자원의 양에 제한이 없습니다.

아이템 접사 link

  • 보호막이 있을 때 행운의 적중 확률: 이제 모든 직업에 나타납니다. 투구에 적용 시 ~12%까지, 목걸이 또는 보조 무기에 적용 시 ~20%까지 감소합니다.

  • 숙련 기술 피해: 신규 추가되었으며 원소술사 무기에 나타납니다. 핵심 기술 피해와 동일하게 증가합니다.
