[파이오니어스 오브 파고니아] 인터페이스 - 음식 생산물 건물 정보

개요 link

세틀러(Settlers) 원작자이자 개발자가 제작하고, 스팀(Steam)에서 **얼리 액세스(Early Access, 앞서 해보기)**로 출시된 도시 건설 시뮬레이션 신작 **파이오니어스 오브 파고니아(Pioneers of Pagonia)**의 인터페이스 중에서 세번째 [음식 생산물] 카테고리에 속하는 건물들의 정보를 정리해보겠습니다.

음식 생산물 link

채집 오두막 link

채집가는 영토에서 채집한 베리류 열매와 버섯은 시장 가판대로, 야생의 아마 섬유는 방직소로 배달합니다.

  • 건설 비용: 무른 나무 몸통
  • 채집품: 라즈베리, 포르치니, 장작, 아마 섬유
  • 인원: 채집가 3명

사냥 오두막 link

사냥꾼은 영토 내에 돌아다니는 동물을 사냥하고 도축해 고기와 가죽을 만들어 냅니다. 고기는 시장 가판대와 식량 공급기로 배달하고, 가죽은 재단사에게 배달합니다.

  • 건설 비용: 무른 나무 몸통
  • 수집품: 토끼 고기, 멧돼지 고기, 사슴 고기, 가죽
  • 인원: 사냥꾼 3명

채소 농장 link

농부는 근처 농지에서 채소를 심고, 물을 주고, 수확합니다. 수확한 채소는 시장 가판대로 배달합니다.

  • 건설 비용: 무른 나무 몸통 5개, 돌 10개
  • 활용: 물
  • 수집품: 호박, 양배추
  • 인원: 농부 3명

작물 농장 link

농부는 근처 농지에서 옥수수, 밀 등의 작물을 심고, 물을 주고, 수확합니다. 밀은 풍차로, 아마 섬유는 방직소로, 옥수수는 시장 가판대로 배달됩니다.

  • 건설 비용: 무른 나무 기둥 10개, 마름 돌 10개
  • 활용: 물
  • 수집품: 옥수수, 아마 섬유, 밀
  • 인원: 농부 3명

농지 link

농지는 채소 농장이나 작물 농장 근처에 배치해야 농부가 농사를 짓습니다. 농지가 위치한 지형에 따라서 더 잘자라는 작물과 덜 자라는 작물이 달라집니다.

우물 link

우물 노동자는 농장, 수목 관리원, 빵집에 물을 공급합니다.

  • 건설 비용: 돌 8개, 밧줄 2개
  • 수집품: 물

풍차 link

풍차에서는 밀을 빻아서 밀가루로 만듭니다.

  • 건설 비용: 무른 나무 기둥 12개, 목재 톱니바퀴 2개, 시멘트 12개, 천 4개
  • 활용: 밀
  • 생산품: 밀가루
  • 인원: 제분업자 3명

빵집 link

빵집에서는 밀가루를 가공해서 빵을 굽습니다. 구운 빵은 시장 가판매와 식량 공급기로 배달됩니다.

  • 건설 비용: 무른 나무 기둥 12개, 시멘트 8개
  • 활용: 밀가루, 물, 장작
  • 생산품: 빵
  • 인원: 음식 노동자 3명
