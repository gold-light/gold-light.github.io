PATH 설정 link

윈도우 10 작업 표시줄 검색 창에 [고급 시스템 설정 보기]라고 입력한 후 나타나는 팝업에서 아래쪽 [환경변수] 버튼을 클릭하면 됩니다.

시작프로그램 관리 link

  • 작업표시줄에서 [마우스 오른쪽 클릭] - [작업관리자]에서 [시작 프로그램] 메뉴에서 관리할 수 있습니다.
  • 키보드의 [Windows 키] + [R]에서 실행할 명령어 입력 칸에 [msconfig]를 입력하고 확인 버튼을 클릭하면 나타나는 [시스템 구성] 팝업에서 [시작 프로그램] 메뉴를 확인하시면 됩니다.
