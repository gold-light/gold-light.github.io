네이버 클라우드 (Ncloud) 자격증 NCA 시험 대비 핵심 요약, 예상문제

개요 link

네이버 클라우드 플랫폼 기반의 기술자격증은 NCA (NAVER CLOUD PLATFORM Certified Associate), NCP (NAVER CLOUD PLATFORM Certified Professional), NCE (NAVER CLOUD PLATFORM Certified Expert)의 3단계로 구성되어 있는데, 그 중에서 가장 기초 단계의 자격증인 NCA 자격증 시험을 준비하면서 공부했던 내용을 시험 날짜를 몇일 앞두고 마지막으로 정리해보려고 합니다.

원래 이렇게 핵심 내용을 정리하는 과정 자체도 시험 준비에 도움이 되기 때문에, 마치 시험 문제의 예상 문제를 뽑아 보는 것 같은 느낌으로 단답형 주관식처럼 정리해보았습니다. (실제 시험은 객관식입니다.)

자격증 상세 내용 link

네이버 클라우드 플랫폼에서 공개하고 있는 NCA 자격증 대한 상세 설명을 정리해보겠습니다.

NCA link

클라우드에 대한 기본적인 이해가 있고 네이버 클라우드 플랫폼을 이용한 인프라 구성이 가능한 수준을 나타낸다고 합니다.
1개 시험(시험코드 100)으로 구성되어 있으며, 시험코드 별 해당 과목은 다음과 같다고 합니다.

  • Overview
  • Compute/Storage/Database
  • Network/Media

핵심 내용 - 예상 문제 link

문제와 답으로 구성했는데 답은 우선은 감추어 두었고, 문제를 클릭하면 아래쪽으로 답이 나타납니다.
(실제 시험은 객관식이지만, 공부를 위해 단답형 주관식으로 정리했습니다.)

4,000 IOPS

- 설명: 표준서버보다 10~20배 이상의 고성능 I/O를 제공하는 SSD가 장착된 서버

GPU Server

- 설명: 현재는 콘솔에서 생성제한된 상태 - 이용하려면 고객센터로 연락해야 함

VDS (Virtual Dedicated Server)

- 설명: 물리 서버 제공 방식이지만 클라우드 서비스 형태로 제공되기 때문에 일반 서버처럼 기존의 네이버 클라우드 플랫폼 서비스와 연동해 서비스를 이용할 수 있음.

Bare Metal Server

- 설명: 물리 서버를 단독으로 제공하는 방식이지만 클라우드 서비스 형태로 사용할 수 있기 때문에 일반 클라우드 서버처럼 네이버 클라우드 플랫폼의 다양한 서비스와 연동하여 이용할 수 있습니다.

High Performance Computing

- 설명: 네이버 클라우드 플랫폼의 전용 HPC 인프라를 이용하여 구성이 가능할 뿐 아니라, 고객이 소유하고 있는 On-Premise 기반의 자원을 이용한 Hybrid 형태의 HPC 환경 구축도 가능. 금융권, 증권사에서 많이 사용함

Data Teleporter

- 설명: 어플라이언스 장비는 기본적으로 고객이 직접 수령하고 반납하는 것을 원칙으로 함.
