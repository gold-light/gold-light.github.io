Windows 환경에서 PHP Composer 설치하는 방법

개요 link

Windows 환경에서 PHP Composer 설치하는 방법을 정리해보았습니다.

Composer란? link

Composer는 php로 개발할때 필요한 패키지간의 의존성을 관리하는 의존성 관리툴이라가 할 수 있습니다. 비슷하게 node.js에는 npm, ruby에는 bundle이 있습니다.

다운로드 link

Composer 공식 사이트에서 Windows용 설치 파일을 다운로드 받습니다.

⁃ composer 다운로드: https://getcomposer.org/download/

설치 link

다운로드 받은 설치 파일을 실행합니다.

  • PC의 모든 계정에서 사용할 수 있게 할 것인지, 현재 계정만 사용할 것인지 선택할 수 있습니다.
  • 개발자 모드를 선택하면 UnInstaller가 포함되지 않습니다.

PHP 위치 설정 link

Composer는 PHP의 패키지 의존성 관리툴이기 때문에 PC에 있는 php.exe 파일의 위치를 지정해주어야 합니다.

  • 위치를 지정한 후 다음 단계로 이동합니다.
  • 혹시 Proxy Server를 사용할 경우 정보를 입력합니다.
  • 준비가 끝났으면 [Install] 버튼을 클릭합니다.
  • 설치가 끝났는데 composer가 제대로 작동하지 않는다면 시스템을 재시작해보시기 바랍니다.
  • 설치가 모두 끝났습니다.

실행 link

설치를 마친 후에 Command 창을 열어서 실행해보면 정상적으로 작동하는 것을 확인할 수 있습니다.
